# Media

This folder will hold the media files of the website. There will be folder for each post that include the media for that post.

## Naming

* Folders will be the same as the post name.

* Files will following syntax:

<item name><-item sequence number><-dimensions(if image is a square add x if not add width and postfix with w)>

**Examples** 
sunflower-01-160x.png
sunflower-01-250w.webp


It will include 2 subfolders folders: images and icons

## images
general purpose images to enrich a page or give more context to the overall page

## icons
similar to images only they are lightweight and used mostly with links.
