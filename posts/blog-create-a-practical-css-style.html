Complete guide to web Fonts

Generic font families

There are 5 generic font families. Each family is a collection of fonts with similar stylistic form.

Generic font family names do not need to be capitalized. The rest do.

Serif

Serif typefaces have decorative slab-like appendages (serifs) on the ends of certain letter strokes. Examples of serif fonts: Times, Times New Roman, Georgia.
Example style:

ABC abc

Sans-serif

Sans-serif typefaces have straight letter strokes that do not end in serifs. Examples of sans-serif fonts: Arial, Arial Black, Verdana. Example style:

ABC abc

Monospace

In Monospace typefaces all characters take up the same amount of space on a line. For example, a capital M will be no wider than a lowercase i. Compare this to proportional typefaces that allot different widths to different characters. Examples: Courier, Courier New, Andale Mono.
Example style:

ABC abc

Cursive

Cursive fonts emulate a script or handwritten style. Examples: Apple Chancery, Zapf-Chancery, Comic Sans.
Example style:

ABC abc

Fantasy

Fantasy fonts are decorative and would be appropriate for headlines and other display type. Fantasy fonts are rarely used for web text due to cross-platform availability and legibility. Examples: Impact, Western.
Example style:

ABC abc

The most used font on the web is Arial followed "Times Roman".

Choose a font color that contrast with the background.
